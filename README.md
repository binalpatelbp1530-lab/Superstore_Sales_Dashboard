This project involves building an interactive and visually appealing Power BI dashboard using the Superstore Sales Dataset. The dashboard provides a comprehensive analysis of sales performance across regions, categories, customers, and time.

Key Objectives:

Analyze overall sales, profit, order volume, and profit margin

Identify high-performing regions and product categories

Track monthly sales trends

Highlight top customers by revenue

Enable interactive filtering using slicers


Tools Used:

Power BI Desktop

Power Query

DAX (Measures)


Key DAX Measures:

Total Sales

Total Profit

Profit Margin (%)

Total Orders


Dashboard Features:
✔ Stylish KPIs
✔ Sales by Region (Bar Chart)
✔ Profit by Category (Donut Chart)
✔ Monthly Sales Trend (Line Chart)
✔ Top 10 Customers (Table)
✔ Filters (Region, Category)
✔ Professional dark theme
✔ Clean layout & formatting

Outcome:
A professional, interactive dashboard that helps business stakeholders understand performance trends and make data-driven decisions
